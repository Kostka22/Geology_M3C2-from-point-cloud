# Geology_M3C2-from-point-cloud
Python GUI tool for M3C2 point cloud change detection. Computes signed distances, significance testing, and LoD between two LAS/LAZ epochs. Exports LAZ, GeoTIFF rasters, quiver plots (PNG/HTML/GPKG), and hillshaded profile cuts. Built on py4dgeo, laspy, NumPy, matplotlib.
